  • Was ist eine Live-CD?

    Eine Live-CD ist ein Betriebssystem, welches in den Speicher geladen wird, dabei werden nur die auf der CD benutzt.
    Es wird nicht die anderen installierten Betriebssysteme oder Daten in den Computer ändern und keine Spur hinterlassen, nach einem Herunterfahren des Computers.
    Es ist auch eine gute Möglichkeit, um ein Betriebssystem ohne Installation auszuprobieren, ein grafischer Weg, um das Betriebssystem zu installieren, während noch die Live-Umgebung benutzt wird, und natürlich eine Möglichkeit, ihr favorisiertes Betriebssystem überall dabei zu haben.

  • Was tun mit der .iso-Datei?

    Eine .iso-Datei ist ein CD- oder DVD-Abbild. Sie können mittels einem Brennprogramm aus ihrem Inhalt eine CD oder DVD erstellen.
    Die daraus entstandene CD darf die .iso-Datei nicht enthalten, oder Sie haben in den Software-Optionen einen Fehler gemacht.
    Sie können ebenfalls die .iso-Datei auf einem USB-Stick extrahieren. Dies wird später beschrieben.

  • Wie Salixlive von CD / DVD booten?

    Vergewissern Sie sich, dass Ihr Computer zuerst versucht, vom optischen Laufwerk zu starten, bevor von Ihrer internen Festplatte gestartet wird.
    Um sicher zu sein, müssen Sie ins Setup vom BIOS (das erste Programm, das vom Computer geladen wird), oft mit der Entf-Taste oder der F2-Taste, und dann ins "Boot-Menü", um die Boot-Reihenfolge zu wählen.
    Wenn diese in Ordnung ist, ist als Einziges nur Salixlive in das optische Laufwerk zu legen und den Computer (neu) zu starten.

  • Wie Salixlive auf USB-Stick installieren?

    1. Öffnen Sie die. .iso-Datei mit einer Entpacker-Software (file-roller, xarchiver, 7-zip, ...).
    2. Extrahieren Sie alle Dateien und Ordner (boot, packages und salixlive) in Ihr USB-Root-Verzeichnis.
    3. Gehen Sie ins "boot"-Verzeichnis auf dem USB-Stick.
    4. Starten Sie "bootinst.sh", wenn Sie in Linux oder Unix sind, "bootinst.bat" wenn Sie unter Windows sind.
      Alternativ können Sie "install-on-USB" starten, wenn Sie unter Linux sind und grub2 installiert haben, das wird den Vorgang „Neustart nach dem ersten Start“ verhindern.

  • Wie vom Salixlive USB-Stick starten?

    In gleicher Weise wie für das Starten vom optischen Laufwerk, müssen Sie sich vergewissern, dass der Start von USB vor Anderem im BIOS-Setup gesetzt ist.
    On the first boot from the USB key, a better and nicer bootloader will be installed (grub2) and the boot process will reboot, it's normal. Beim ersten Start vom USB-Stick wird ein besserer und schönerer „Bootloader“ installiert (grub2) und es wird neu gestartet werden, das ist normal.

  • Wie Veränderungen trotz Neustart behalten?

    Eine Anwendung (create-slxsave-xfs) ist beim Salixlive Desktop enthalten. Starten Sie diese und befolgen Sie die Schritte zum Erstellen einer Datei, die Ihre Änderungen trotz Neustart behält.
    Diese Datei wird automatisch beim nächsten Neustart verwendet werden, und so beginnen, jede Änderung zu überwachen und zu speichern.
    Um die Änderungen zu entfernen und wieder auf Standardeinstellungen zurückzukommen, löschen Sie einfach die Datei slxsave.xfs.

  • Wie Salix aus Salixlive installieren?

    Einige Anwendungen (gparted, salixlive-installer, lilosetup) sind beim Salixlive Desktop enthalten.
    gparted sollte genutzt werden, um Partitionen auf der Ziel-Festplatte zu erstellen.
    Dann führen Sie salixlive-installer aus und folgen den Schritten, um Salix auf Ihrer Partition zu installieren.
    Ein „Bootloader“ (ein Programm für das Starten mehrerer Betriebssysteme) wird installiert werden, wenn Sie es wünschen (sehr empfehlenswert).
    Hinweis: Salix wird installiert werden, nicht die Live-Umgebung. Es gibt keinen Unterschied zur Installation von Salix aus einer Installations-CD.

  • Τι είναι ένα Live CD;

    Ένα Live (ζωντανό) CD είναι ένα λειτουργικό σύστημα που φορτώνεται στη μνήμη χρησιμοποιώντας μόνο τα δεδομένα που υπάρχουν στο CD.
    Δεν θα τροποποιήσει άλλα λειτουργικά συστήματα που είναι εγκατεστημένα ή δεδομένα στον Η/Υ και δεν θα αφήσει καθόλου ίχνη μετά τον τερματισμό του Η/Υ.
    Είναι επίσης ένας ιδανικός τρόπος να δοκιμάσετε ένα λειτουργικό σύστημα χωρίς να το εγκαταστήσετε, ένας γραφικός τρόπος να εγκαταστήσετε το λειτουργικό σύστημα ενώ είστε στο ζωντανό περιβάλλον, και επίσης ένας τρόπος να έχετε το αγαπημένο σας λειτουργικό σύστημα, παντού, μαζί σας.

  • Τι να κάνω με το αρχείο .iso;

    Ένα αρχείο .iso είναι μια εικόνα ενός CD ή DVD. Μπορείτε να δημιουργήσετε ένα CD ή DVD με αυτό χρησιμοποιώντας λογισμικό που καίει την εικόνα στο CD ή DVD.
    To CD που θα δημιουργηθεί δεν πρέπει να περιέχει το αρχείο .iso, διαφορετικά έχετε κάνει λάθος στις επιλογές του προγράμματος δημιουργίας του CD ή DVD.
    Μπορείτε επίσης να εξάγετε τα περιεχόμενα του αρχείου .iso σε ένα USB. Αυτό συζητείτε παρακάτω.

  • Πως να εκκινήσετε (μπουτάρετε) με το Salixlive CD/DVD;

    Σας διαβεβαιώνω οτι ο Η/Υ θα προσπαθήσει κατ' αρχήν να εκκινήσει (μπουτάρει) από τη συσκευή CD/DVD και κατόπιν από το σκληρό δίσκο.
    Για να είστε σίγουροι πρέπει να πάτε στις ρυθμίσεις του BIOS (το πρώτο πρόγραμμα που φορτώνεται από τον Η/Υ), χρησιμοποιώντας συνήθως το Del-key ή το F2-key και κατόπιν πηγαίνετε στο "boot menu" για να επιλέξετε τη σειρά εκκίνησης.
    Εάν η σειρά είναι εντάξει, το μόνο πράγμα που χρειάζεται να κάνετε είναι να τοποθετήσετε το Salixlive στη συσκευή CD/DVD και να εκκινήσετε ή να επανεκκινήσετε τον Η/Υ. Διαφορετικά επιλέξτε σαν πρώτη συσκευή εκκίνησης το CD/DVD.

  • Πως να εγκαταστήσετε το Salixlive στο USB key;

    1. Ανοίξτε το αρχείο .iso με ένα διαχειριστή συμπιεσμένων αρχείων (file-roller, xarchiver, 7-zip, ...).
    2. Αποσυμπιέστε (εξάγετε) όλα τα αρχεία και τους φακέλους (boot, packages and salixlive) στο ριζικό (root) κατάλογο του USB σας.
    3. Πηγαίνετε στο κατάλογο "boot" του USB key.
    4. Εκτελέστε το "bootinst.sh" εάν είστε στο Linux ή Unix, το "bootinst.bat" εάν είστε στα Windows.
      Εναλλακτικά, μπορείτε να εκτελέσετε το "install-on-USB" εάν είστε στο Linux και έχετε το grub2 εγκατεστημένο. Αυτό θα εμποδίσει τη πρώτη λειτουργία "επανεκκίνηση στη πρώτη εκκίνηση" ("reboot on first boot").

  • Πως να εκκινήσετε (μπουτάρετε) με το Salixlive USB key;

    Κατά τον ίδιο τρόπο για να εκκινήσετε από τη συσκευή CD/DVD, πρέπει να επιβεβαιώσετε ότι η εκκίνηση από το USB είναι πρώτη, πριν τις άλλες, στις ρυθμίσεις του BIOS.
    Στη πρώτη εκκίνηση από το USB key, ένας καλύτερος και πιο φιλικός bootloader θα εγκατασταθεί (grub2) και η διαδικασία εκκίνησης θα επανεκκινήσει. Αυτό είναι φυσιολογικό.

  • Πως να διατηρήσετε τις αλλαγές μετά την επανεκκίνηση;

    Ένα λογισμικό (create-slxsave-xfs) περιλαμβάνεται στην επιφάνεια εργασίας του Salixlive. Εκτελέστε το και ακολουθήστε τα βήματα για να δημιουργήσετε ένα αρχείο που θα κρατήσει τις αλλαγές σας μετά την επανεκκίνηση.
    Αυτό το αρχείο θα χρησιμοποιηθεί αυτόματα στην επόμενη επανεκκίνηση και έτσι θα αρχίσει να παρακολουθεί και να σώζει τις όποιες αλλαγές.
    Για να απομακρύνετε τις όποιες αλλαγές και να γυρίσετε πίσω στις εξ΄ορισμού ρυθμίσεις, απλά διαγράψτε το αρχείο slxsave.xfs.

  • Πως να εγκαταστήσετε το Salix από το Salixlive;

    Μερικά λογισμικά (gparted, salixlive-installer, lilosetup) περιλαμβάνονται στην επιφάνεια εργασίας του Salixlive.
    Το gparted θα πρέπει να χρησιμοποιηθεί για τη δημιουργία διαμερισμάτων στο σκληρό δίσκο που θα γίνει η εγκατάσταση.
    Κατόπιν εκτελέστε το salixlive-installer και ακολουθήστε τις οδηγίες για να εγκαταστήσετε το Salix στο διαμέρισμά σας.
    Ένας εκκινητής (bootloader) (ένα πρόγραμμα για την εκκίνηση πολλαπλών λειτουργικών συστημάτων) θα εγκατασταθεί εάν το επιθυμείτε (προτείνεται ανεπιφύλακτα).
    Σημείωση: Το Salix θα εγκατασταθεί, όχι το ζωντανό περιβάλλον (live environment). Δεν υπάρχει καμμιά διαφορά από την εγκατάσταση του Salix από ένα CD εγκατάστασης.

  • What is a Live CD?

    A Live CD is a operating system which loads into memory using only the data provided on the CD.
    It will not modify the other operating systems installed or data in the computer and will leave no trace after a computer shutdown.
    It is also a good way to discover an operating system without installation, a graphical way to install the operating system while still using the live environment, and of course a way of having its favourite OS with you everywhere.

  • What to do with the .iso file?

    An .iso file is a CD or DVD image. You can make a CD or DVD with it using a burning software.
    The resulting CD must not contain the .iso file, or you made a mistake in the software options.
    You can also explode the .iso file on an USB key. This is discuss further.

  • How to boot on Salixlive CD/DVD?

    You have to assure you that your computer will first try to boot on the optical drive before booting on your internal disk.
    To be sure, you need to go the BIOS (the first program loaded by the computer) setup, often using the Del-key or the F2-key, and then go in the "boot menu" to choose the boot ordering.
    If this is ok, the only thing to do is putting Salixlive in the optical drive and start or restart the computer.

  • How to install Salixlive on USB key?

    1. Open the .iso file using an archiver software (file-roller, xarchiver, 7-zip, ...).
    2. Extract all files and folders (boot, packages and salixlive) to your USB root directory.
    3. Go the "boot" directory on the USB key.
    4. Run "bootinst.sh" if you're on Linux or Unix, "bootinst.bat" if you're on Windows.
      Alternatively, you can run "install-on-USB" if you're on Linux and have grub2 installed ; this will prevent the first "reboot on first boot" operation.

  • How to boot on Salixlive USB key?

    In the same manner to boot from the optical drive, you must assure you that the USB boot is set prior others in the BIOS setup.
    On the first boot from the USB key, a better and nicer bootloader will be installed (grub2) and the boot process will reboot, it's normal.

  • How to keep changes uppon reboot?

    A software (create-slxsave-xfs) is included on the Salixlive desktop. Run it and follow the steps to create a file that will keep your changes uppon reboot.
    This file will automatically be used on the next reboot, and so will begin to monitor and save any change.
    To remove any changes and come back to the default settings, simply delete the slxsave.xfs file.

  • How to install Salix from Salixlive?

    Some softwares (gparted, salixlive-installer, lilosetup) are included on the Salixlive desktop.
    gparted should be used to create partitions on the target hard disk.
    Then run salixlive-installer and follow the steps to install Salix on your partition.
    A bootloader (a program to boot multiple operating systems) will be installed if you wish (highly recommended).
    Note: Salix will be installed, not the live environment. There is no difference from installing Salix from a installation CD.

  • Qu'est-ce qu'un Live CD ?

    Un Live CD est un système d'exploitation qui se charge en mémoire en utilisant uniquement les données présentes sur le CD.
    Ça ne modifiera pas les autres systèmes d'exploitation installés ou les données de l'ordinateur et ne laissera aucune trace après un arrêt de l'ordinateur.
    C'est également un bon moyen de découvrir un système d'exploitation sans installation, une manière graphique d('installer le système d'exploitation tout en continuant d'utiliser l'environnement live, et bien sûr un moyen d'avoir son SE favoris avec soit partout.

  • Quoi faire avec le fichier .iso ?

    Un fichier .iso est une image CD ou DVD. Vous pouvez en faire un CD ou un DVD avec en utilisant un logiciel de gravure.
    The CD résultant ne doit pas contenir le fichier .iso, ou vous avez fait une erreur dans les options du logiciel.
    Vous pouvez aussi éclater le fichier .iso sur une clé USB. Ceci est abordé plus loin.

  • Commnet démarrer sur un CD/DVD de Salixlive ?

    Vous devez vous assurer que votre ordinateur va d'abbord essayer de démarrer sur le lecteur optique avant de démarrer sur votre disque interne.
    Pour en être sûr, vous devez vous rendre dans le setup du BIOS (le premier programme chargé par l'ordinateur), bien souvent en utilisant la touche Suppr. ou la touche F2, et enfin dans le le menu de démarrage ("boot menu" en anglais) pour choisir l'ordre de démarrage.
    Si c'est bon, la seule chose à faire est d'insérer Salixlive dans le lecteur optique et démarrer ou redémarrer l'ordinateur.

  • Comment installer Salixlive sur une clé USB ?

    1. Ouvrir le fichier .iso en utilisant un logiciel de gestion d'archives (file-roller, xarchiver, 7-zip, ...).
    2. Extraire tous les fichiers et répertoires (boot, packages and salixlive) vers le répertoire racine de votre clé USB.
    3. Aller dans le répertoire "boot" de la clé USB.
    4. Exécuter "bootinst.sh" si vous êtes sur Linux ou Unix, "bootinst.bat" si vous êtes sur Windows.
      De manière alternative, vous pouvez exécuter "install-on-USB" si vous êtes sur Linux et que grub2 est installé ; ceci évitera l'opération de « redémarrage au premier démarrage ».

  • Comment démarrer Salixlive sur la clé USB ?

    De la même manière que pour démarrer sur le lecteur optique, vous devez vous assurer que le démarrage sur USB est prioritaire sur les autres dans les setup du BIOS.
    Lors du premier démarrage sur la clé USB, a meilleur et plus joli chargeur de démarrage (bootloader) sera installé (grub2) et le processus de démarrage va redémarrer l'ordinateur, ceci est normal.

  • Comment garder les changements après un redémarrage ?

    Un logiciel (create-slxsave-xfs) est inclu sur le bureau de Salixlive. Exécutez-le et suivez les étapes pour créer un fichier qui gardera vos changements après redémarrage.
    Ce fichier sera automatiquement utilisé au prochain redémarrage, et commencera donc à enregistrer chaque chanegement.
    Pour supprimer tous les changements et revenir à la configuration par défaut, il suffit de supprimer le fichier slxsave.xfs.

  • Comment installer Salix depuis Salixlive ?

    Certains logiciels (gparted, salixlive-installer, lilosetup) sont inclus sur le bureau de Salixlive.
    gparted doit être utilisé pour créer les partitions sur le disque destination.
    Lancez alors salixlive-installer et suivez les étapes pour installer Salix dans votre partition.
    Un chargeur de démarrage (un programme pour démarrer plusieurs systèmes d'exploitation) sera installé si vous le désirez (fortement conseillé).
    Note : C'est Salix qui sera installé et non l'environnement live. Il n'y a aucune différence par rapport à l'installation de Salix depuis un CD d'installation.
